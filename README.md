# Villám Meló 

**2025/2026-os Vizsgaremek Projekt**

**Weboldal:** [www.villammelo.hu](https://www.villammelo.hu/)

---

##  A projektről (Miért pont ezt választottuk?)

A mai felgyorsult világban egyre nagyobb igény mutatkozik a rugalmas munkavégzésre és a gyors jövedelemszerzési lehetőségekre. 
Bár léteznek szakembereket közvetítő portálok, hiányzik egy olyan platform, amely a képesítést nem igénylő, egyszerűbb feladatok elvégzésére köti össze a keresletet a kínálattal. 
A "Villám Meló" projekt lehetőséget biztosít a felhasználóknak a gyors feladatmegosztásra és jövedelemszerzésre.

---

## Főbb funkciók (Tervezett / Megvalósított)
* **Felhasználói profilok:** Személyes és általános adatok tetszés szerint változtathatóak.
* **Gyors hirdetésfeladás:** Feladatok feltöltése kategóriák, helyszín és díjazás megadásával.
* **Keresés és szűrés:** A munkavállalók könnyedén találhatnak, számukra ideális feladatokat.
* **Jelentkezés:** Egyszerű és gyors jelentkezési folyamat az adott munkára.
* **Értékelés:** Értékelni és véleményezni lehet profilokat.

## Használt technológiák és eszközök

A projekt során a klasszikus webes technológiákra támaszkodtunk:

* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Backend:** PHP
* **Adatbázis:** MySQL
* **Eszközök & Verziókövetés:** WinSCP, Git / GitHub
